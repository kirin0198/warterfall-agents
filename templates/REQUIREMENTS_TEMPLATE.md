## 使い方

1. 上記の各セクションを記入する（わかる範囲で OK）
2. このファイルの内容をコピーして、Claude Code で以下のように伝える:

```
以下の要件で仕様を作ってください。

{テンプレートの内容を貼り付け}
```

または、ウォーターフォールフローで開始する場合:

```
以下の要件でウォーターフォール開発を始めてください。

{テンプレートの内容を貼り付け}
```
